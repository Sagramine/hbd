# Hi Darling!
I made this for you.

```bash
#################################
# Install required dependencies #
#################################
$ yarn
# OR #
$ npm install

##############################################################
# Serve with hot reload at {host}:{port} from config.js file #
##############################################################
$ yarn dev
# OR #
$ npm run dev

#####################
# For deploying, do #
#####################
$ yarn build
# OR #
$ npm run build
```
